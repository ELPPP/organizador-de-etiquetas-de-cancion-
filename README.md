Este proyecto fue concebido originalmente como una herramienta para automatizar y estandarizar la organización de los metadatos (etiquetas) de los archivos de música MP3.
El objetivo era simplificar la gestión de una gran biblioteca musical.

Actualmente, el repositorio contiene un script funcional en Python que extrae los títulos de las canciones y otros metadatos básicos y los exporta a un archivo de Excel
para su posterior revisión manual.

Aunque el objetivo inicial no se completó, el proyecto sirve como un registro de los esfuerzos y las ideas. Se consideró la posibilidad 
de integrar APIs de servicios de música como Spotify o YouTube o utilizar inteligencia artificial para obtener metadatos y aplicar una organización más inteligente.
El desarrollo se detuvo debido a limitaciones de tiempo y conocimiento técnico en el momento.
